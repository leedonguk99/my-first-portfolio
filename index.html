프로젝트

커피 AI 추천 시스템

사용자의 취향 데이터를 학습해, 로스팅·산지·풍미 프로파일 기반으로 개인 맞춤형 커피를 추천하는 AI 커피 큐레이션 앱

React Native Python AI/ML 빅데이터

1️⃣ 문제 정의

  • 커피 선택 기준이 전문적이고 복잡함
    로스팅, 산지, 프로세싱, 향미 노트 등 다양한 요소로 인해 초보자·중급자 모두 "내 취향에 맞는 커피를 고르기 어렵다"는 문제
  • 기존 추천의 한계
    단순 인기순이나 전문가 기준 중심으로 개인 취향 반영 부족

2️⃣ 해결 방안

  • 사용자의 선호 데이터를 구조화
  • 커피 데이터를 객관적 지표로 벡터화
  • AI 알고리즘을 통해 취향 기반 맞춤 추천 시스템 제공

👉 결과적으로 "커피를 공부하지 않아도, 취향에 맞는 커피를 발견할 수 있는 경험"을 제공

3️⃣ 핵심 기능

① 사용자 취향 수집
  • 선호 로스팅 단계
  • 좋아하는 풍미(산미/단맛/바디감 등)
  • 이전 커피 평가(별점, 피드백)
② 커피 데이터베이스
  • 로스팅 정도
  • 생산지·품종
  • 프로세싱 방식
  • 향미 프로파일
③ AI 추천 로직
  • 협업 필터링 + 콘텐츠 기반 추천 혼합
  • 사용자 취향과 커피 특성 간 유사도 분석
  • 사용 이력에 따라 추천 결과 지속 학습
④ 추천 결과 제공
  • "당신과 87% 유사한 커피"
  • 추천 이유 시각화 (ex. 산미 선호 + 에티오피아 내추럴)

4️⃣ 기술적 접근 방식

빅데이터 분석
  • 커피 속성 데이터 정규화
  • 사용자 행동 데이터 축적
AI / ML
  • 취향 벡터화
  • 추천 알고리즘 적용
확장성 고려
  • 신규 커피·로스터리 데이터 추가 가능
  • 추천 정확도 지속 개선

👉 "단순 추천 앱이 아니라 학습하는 시스템"이라는 점을 강조

5️⃣ 기대 효과

사용자 관점
  • 실패 없는 커피 선택
  • 취향의 언어화 → 커피 이해도 상승
산업 관점
  • 로스터리·카페와의 연계 가능
  • 개인화 기반 커피 마케팅 플랫폼으로 확장
