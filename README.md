# 포트폴리오 웹사이트

간단하고 현대적인 포트폴리오 웹사이트입니다.

## 기능

- 반응형 디자인 (모바일, 태블릿, 데스크톱 지원)
- 부드러운 스크롤 네비게이션
- 섹션별 콘텐츠 (홈, 소개, 기술, 프로젝트, 연락처)
- 스크롤 애니메이션 효과

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열면 됩니다.
2. 또는 로컬 서버를 실행하여 사용할 수 있습니다:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server 사용)
   npx http-server
   ```

## 커스터마이징

- `index.html`: 콘텐츠 수정
- `styles.css`: 스타일 및 색상 변경
- `script.js`: 기능 추가 또는 수정

## 파일 구조

```
my-first-portfolio/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 파일
└── README.md       # 프로젝트 설명
```
